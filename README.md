# calculadoraweb

##  1.HTML 5
Para contenido o estructura. 
## 2. CSS
Para diseño y estilos 
## 3. Javascript 
Para las funciones (backend)

Creado por Mayra Yépez, en el Bootcamp de Tecnolochicas PRO, Mexico.
